# Jouw eigen website

Je hebt geleerd hoe HTML, CSS en JavaScript werken. Nu is het tijd om iets echts te maken — een website die je zelf bouwt, in een echte editor, en die je kunt delen met anderen.

## De editor

Op **editor.coderius.nl** kun je direct aan de slag. Je schrijft HTML, CSS en JavaScript in een volwaardige online editor en ziet het resultaat meteen. Geen installatie nodig.

[Ga naar editor.coderius.nl →](https://editor.coderius.nl/)

## Wat kun je bouwen?

Geen verplichte opdracht — dit is jouw project. Een paar ideeën:

- Een pagina over jezelf (wie ben je, wat vind je leuk?)
- Een overzicht van je favoriete films, games of muziek
- Een kleine quiz met knoppen en JavaScript
- Een pagina voor een bedrijf, band of evenement dat je verzint

## Tips

- Begin klein. Eén kop, één paragraaf, één afbeelding. Bouw daarna verder.
- Gebruik de [Cheatsheet](/cheatsheet) als je een CSS-eigenschap of HTML-tag even niet meer weet.
- Experimenteer. Als je iets probeert en het werkt niet, lees de foutmelding en probeer opnieuw.
