# JavaScript Basics

JavaScript is de taal die websites interactief maakt. Waar HTML de structuur bepaalt en CSS de opmaak, zorgt JavaScript voor het gedrag — knoppen die reageren, tekst die verandert, en animaties die bewegen.

## Hoe te beginnen?

Volg de lessen hieronder in volgorde. Elke les bouwt voort op de vorige en heeft een interactieve editor om direct mee te oefenen.
